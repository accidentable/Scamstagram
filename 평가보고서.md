# ScamKeep 프로젝트 평가 보고서

## 프로젝트 개요
**ScamKeep**은 AI 기반의 스캠/피싱 예방 및 커뮤니티 보상 플랫폼입니다. 갈수록 교묘해지는 피싱·스캠 범죄에 대응하기 위해 시민들의 자발적인 신고와 공유를 유도하며, 지속 가능한 보안 커뮤니티를 형성합니다.

#### 2.2 AI 활용 방식에서 드러나는 독창적 아이디어

**Gemini AI 활용 방식:**
```
[입력] 스캠 의심 이미지 (스크린샷)
          ↓
[AI 분석] Google Gemini 2.0-Flash
          ↓
[출력]
  • isScam: 스캠 여부 (boolean)
  • confidenceScore: 신뢰도 (0-100)
  • scamType: 스캠 유형 (13가지 분류)
  • riskLevel: 위험도 (LOW/MEDIUM/HIGH/CRITICAL)
  • extractedTags: 자동 태그 추출
  • analysis: 한국어 설명 (2-3문장)
```

**독창적 요소:**
1. **멀티모달 AI 분석**: 텍스트가 아닌 이미지 직접 분석

4. **AI 자동 설명 생성**: 분석 결과를 SNS 게시물로 자동 작성

**평가**: AI를 단순 분류가 아닌 커뮤니티 콘텐츠 생성까지 연결한 점이 독창적

**평가**: "보안 = 재미없음"이라는 고정관념을 깬 게이미피케이션 접근이 참신
